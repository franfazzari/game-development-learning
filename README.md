# Game Development Learning

A hands-on journey into building a game engine from scratch, progressing toward understanding modern game engine tools and architectures.

## Phase 1 — Software Rasterizer

Build a fully functional 3D software rasterizer from scratch in C++, implementing every step of the rendering pipeline manually on the CPU.

| Phase | Topic | Status |
|-------|-------|--------|
| 1 | Setup & Window | ⬜ |
| 2 | Math Foundation | ⬜ |
| 3 | 2D Primitives | ⬜ |
| 4 | 3D Pipeline | ⬜ |
| 5 | Depth & Visibility | ⬜ |
| 6 | Shading & Textures | ⬜ |
| 7 | Scene & Polish | ⬜ |

Full curriculum: [notes/curriculum.md](notes/curriculum.md)
