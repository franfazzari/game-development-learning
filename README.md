# Game Development Learning

A hands-on journey into building a game engine from scratch, progressing toward understanding modern game engine tools and architectures.

## Roadmap

### Phase 1 — Engine From Scratch
Build a game engine piece by piece, starting from the lowest level:
- Window creation & OS abstraction
- Rendering pipeline (CPU rasterizer → OpenGL → Vulkan)
- Math library (vectors, matrices, transforms)
- Entity-Component System (ECS)
- Asset loading (meshes, textures)
- Input handling
- Basic physics

### Phase 2 — Modern Tools & Pipelines
Understand how production engines work:
- Engine architecture deep-dives (Unreal, Godot source)
- Shader systems & material graphs
- Scene editors & tooling
- Scripting integration
- Profiling & optimization

## Structure

